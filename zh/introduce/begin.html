# 前端应用

1.从远程仓库拉取代码

git clone http://106.15.88.88/vms/vms-react.git

2.拉取依赖包并启动项目

yarn install    // 安装依赖
npm start       // 启动应用

注意

如果启动项目后报错node-sass的问题， 可以用npm rebuild 重装一下node-sass， 再启动项目。
